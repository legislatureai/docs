**Tags** are an easy way to sort bills and other matters into categories.

## The Tags page

To get started managing tags, go to the tags page using the link on the Insights navigation bar at the top left of the page.

<Frame>
  <img src="/images/bartag.png" style={{ borderRadius: "0.5rem" }} />
</Frame>

There you'll see any tags you have already created, along with a 'Create tag' button:

<Frame>
  <img src="/images/tagpage.png" style={{ borderRadius: "0.5rem" }} />
</Frame>

## Creating tags

Click 'Create tag' on the tags page to start creating a tag. Tags can be assigned a name, short description, and highlight color.

<Frame>
  <img src="/images/createtag.png" style={{ borderRadius: "0.5rem" }} />
</Frame>

Once a tag is created, you can delete it or update its name, description, or color with the buttons on the tag's card:

<Frame>
  <img src="/images/tag.png" style={{ borderRadius: "0.5rem" }} />
</Frame>

## Tagging and Untagging Bills

To get started tagging bills, navigate to any bill's page. You can find bills by visiting the [bills page](https://legislature.ai/bills).

When you are logged in as an Insights user, you will see a 'Tags' section at the top of the left-hand column. Click the triple dots at the top right of that box and select 'Add Tag' to add a tag:

<Frame>
  <img src="/images/tagbill.png" style={{ borderRadius: "0.5rem" }} />
</Frame>

Once a tag is applied to a bill, you can click the 'x' on the right of the shown tag to remove that tag from the bill:

<Frame>
  <img src="/images/untagbill.png" style={{ borderRadius: "0.5rem" }} />
</Frame>

## Filtering by Tags

Once you have tagged bills, you can apply the tag's filter on any of the main pages (the [meetings page](https://legislature.ai/meetings), [bills page](https://legislature.ai/bills), or [news page](https://legislature.ai/news)).

On the left sidebar, find the Tags filter and click to select your tag:

<Frame>
  <img src="/images/tagfilter.png" style={{ borderRadius: "0.5rem" }} />
</Frame>

Once a tag is selected, the results will drill down to only the bills tagged (or only meetings where those bills are scheduled if you're on the meetings page, or news articles related to those bills if you're on the news page).

<Frame>
  <img src="/images/tagselected.png" style={{ borderRadius: "0.5rem" }} />
</Frame>

The tags filter can be combined with other filters to drill down further as well.
